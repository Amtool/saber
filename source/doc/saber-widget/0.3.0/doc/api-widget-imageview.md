# ImageView

图片查看器控件。


## Usage

``` javascript
var widget = require( 'saber-widget' );
require( 'saber-widget/plugin/Masker' );
require( 'saber-widget/ImageView' );

var imageview = widget.imageview( element );
console.info( imageview.get( 'items' ) );
```
