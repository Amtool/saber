saber-emitter

一个适用于移动端的事件发射器，兼容 node 环境。

或者在 node 环境下 使用 npm 引入模块：

Usage

API

创建 Emitter 实例

  • return {Emitter}

Methods

mixin(obj)

通过 mixin 将 Emitter 混入目标对象

  • obj {Object} 目标对象
  • return {Object} 混入 Emitter 后的目标对象

Classes

on(event, listener)

挂载事件

  • event {string} 事件名
  • listener {Function} 监听器
  • return {Emitter}

once(event, listener)

挂载只执行一次的事件

  • event {string} 事件名
  • listener {Function} 监听器
  • return {Emitter}

off([event[, listener]])

注销事件与监听器

  • event {string=} 事件名
  • listener {Function=} 监听器
  • return {Emitter}

emit(event[, …args])

触发事件

  • event {string} 事件名
  • args {...*} 传递给监听器的参数，可以有多个
  • return {Emitter}

listeners(event)

返回指定事件的监听器列表

  • event {string} 事件名
  • return {Array} 监听器列表

setMaxListeners(number)

设置每个事件下，监听器的最大个数。为 0 时不限制，默认值是 10

  • number {number} 监听器个数
  • return {Emitter}
