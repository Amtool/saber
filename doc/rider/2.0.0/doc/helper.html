辅助函数

file-exists(file)

检测 文件路径 是否存在。

相关：图片，排版

is-url(url)

检测是否是 URL。

这里的检测并不严格，此函数的目的主要是快速区分 远程路径 与 本地路径。

相关：排版

get-retina-src(src, dppx, separator)

获取高分辨率版本的图片地址。

  • src - 图片路径
  • dppx - 图片尺寸的倍数
  • separator (可选) - 分隔符，默认为 @

相关：图片
